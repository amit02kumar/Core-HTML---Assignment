List of Tags.

  1. h1: Defines a heading level 1.
  2. p: Defines a paragraph of text.
  3. a: Defines a HyperLink
  4. div: Defines a container for HTML elements.
  5. b: Defines Bold Text
  6. br: Defines a single line break
  7. table: Defines a table with rows and columns.
  8. ul: Defines an unordered list.
